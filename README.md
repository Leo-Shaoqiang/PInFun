
## 体验项目

克隆本仓库，然后进入 PInFun:

```bash
git clone https://github.com/MooozzZ/PInFun.git
cd PINFUN
```

安装 npm 包：

```bash
npm install
# yarn
```

开启微信小程序开发服务器：

```bash
npm run dev:weapp
# yarn dev:weapp
```

开启支付宝小程序开发服务器：

```bash
npm run dev:alipay
# yarn dev:alipay
```
